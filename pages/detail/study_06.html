H&M 쇼핑몰
리액트를 통해 H&M 쇼핑몰을 만들었습니다.
Redux, react router dom을 사용하여 검색 기능 및 상품 상세페이지 작업을 진행하셔으며, redux toolkit으로 저장소를 설정하여 유지보수 하기 쉽게 작업했습니다. 전체적인 디자인은 bootstrap과 css를 사용하여 반응형으로 작업했습니다.
  • React
  • Redux
  • React Router Dom
  • Redux Toolkit
  • Json Server
  • Bootstrap
설명
  • 상품 리스트를 볼 수 있습니다.
  • 원하는 상품을 검색할 수 있습니다.
  • 상품 클릭시 해당 상품 상세 페이지로 이동됩니다.
  • 상품 상세페이지에서 해당 상품에 옵션을 선택할 수 있습니다.
