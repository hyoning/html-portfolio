뉴스

유료 뉴스 API, 무료 유투브 API, 무료 날씨 API를 사용하여 제작하였으며, reactQuery를 사용하여 데이터 및 캐싱, 동기화 작업을 하였습니다. Firebase를 통해 아이디, 비번 및 북마크한 데이터를 저장하였습니다.
*유료 API 로 작업되어 해당 페이지는 확인 불가합니다.

설명
  • 뉴스 메인 페이지에서 나라별, 언론사별 뉴스와 현재 날씨, 유투브 뉴스 영상을 확인할 수 있습니다.
