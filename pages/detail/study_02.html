숫자 맞추기 게임
바닐라 자바스크립트를 통해 게임을 만들었습니다.
숫자 맞추기 게임은 if문의 대해 이해하기 쉬운 프로젝트입니다.
  • javascript
  • html
  • css
설명
  • 1~50 사이에 숫자 입력할 수 있습니다.
  • 정답보다 클 때 "down!!" 문구가 출력됩니다.
  • 정답보다 작을 때 "UP!!" 문구가 출력됩니다.
  • 정답일 때 "맞추셨습니다!!" 문구가 출력됩니다.
  • 기회 다 썼을 때 "다시 도전하세요!! 정답은 00" 문구가 출력됩니다.
  • 리셋 버튼 클릭시 "좋아! 이번엔 꼭 성공해" 문구가 출력됩니다.
  • 1보다 작거나 50보다 큰 수를 입력시 "1과 50사이 숫자를 입력해주세요." 문구가 출력됩니다.
  • 동일한 값을 입력했을 때 "이미 입력한 숫자입니다. 다른 숫자를 입력해주세요." 문구가 출력됩니다.
  • 몇번의 기회가 남아있는 지 문구가 출력됩니다.
  • 에러 문구 출력 시 기회가 줄지 않습니다.
  • go 버튼 클릭 시, 정답일 시, 기회 소진 시, reset 버튼 클릭 시 효과음이 나옵니다.
